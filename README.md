# Healthcare Management System

Java Swing MVC app (VS Code friendly) using **all 7 CSV files** in `/data`.

## Key behaviour
- Loads CSV on startup.
- CRUD operations using the files along with export.
- Refresh button (shows new + old) data in csv files.
- Referral + Prescription export to TXT in `/output`.
